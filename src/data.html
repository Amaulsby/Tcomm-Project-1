How to organize and analyze research data

For this lesson, you'll need:

  • Microsoft Excel
  • PASW (formerly SPSS) or a free trial

You should already be familiar with:

  • Excel basics
  • Descriptive and inferential statistics
  • Research design & variable identification

This lesson is intended for graduate or advanced undergraduate students working on original research. By the end, you’ll be able to:

  • Format raw data in Excel for analysis
  • Run basic descriptive statistics in Excel
  • Prepare your dataset for use in PASW

Format data for analysis

Open the Excel file ToolBox_DataLesson.xls. In the Raw_Dataset sheet:

  • Ensure variable names are unique and intuitive
  • Rename variables starting with numbers (e.g., 4tutorial_types to tutorial_types)
  • Use descriptive variable names (e.g., use to perc_ease_of_use)

Clean data by using consistent formatting. For example:

  • Change "female" to "F" in gender
  • Break compound values (e.g. Y/Y/N) into multiple columns
  • Delete redundant columns

Use 0/1 for categorical data to allow better testing in PASW.

Dichotomous variables help isolate characteristics for statistical testing.

Copy the data to Sheet2 and rename it Formatted_Data. Use Insert > Columns to expand categorical columns as needed and apply 0/1 encoding. Use Edit > Replace to make substitutions.

Run descriptive statistics

Descriptive stats help identify new patterns and suggest new questions.

  • Check that perc_enjoy, perc_usefulness, and perc_ease_of_use are in adjacent columns
  • Enable the Analysis ToolPak under Tools > Add-Ins
  • Run Descriptive Statistics from Tools > Data Analysis
  • Set input range, uncheck labels, and enable summary statistics

Run inferential statistics

Open PASW and load the Excel file (Formatted_Data sheet). Go to Analyze to access statistical tools. Try to locate:

  • Wilcoxon-Mann-Whitney test
  • Simple linear regression
  • Non-parametric correlation

Consult UCLA’s Choosing the Correct Statistic to select appropriate tests based on your question and data type.

To compare perc_usefulness across free_play types, use:

  1. Analyze > Nonparametric Tests > K Independent Samples...
  2. Select perc_usefulness as the dependent variable
  3. Select free_play as the grouping variable
  4. Define range: Min = 0, Max = 1

Click OK to view results. Export the output (Word, Excel, or PDF) via File > Export.
